# Secure Login System

## Project Overview
This project is a secure login web application developed using Python Flask.

## Features
- User Registration
- Secure Login
- Password Hashing using bcrypt
- SQLite Database
- SQL Injection Protection
- Session Management
- Logout Feature
- Responsive User Interface

## Technologies Used
- Python
- Flask
- Flask-Bcrypt
- SQLite
- HTML
- CSS

## Installation

pip install -r requirements.txt

python database.py

python app.py

Open:
http://127.0.0.1:5000

## Security Features
- Passwords are stored as bcrypt hashes.
- SQL queries use parameterized statements.
- User sessions are securely managed.
- Duplicate email registration is prevented.
